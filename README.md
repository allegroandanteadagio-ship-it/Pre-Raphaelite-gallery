# 🎨 Галерея картин прерафаэлитов

Django-веб-приложение для отображения коллекции картин прерафаэлитов с адаптивным дизайном и функцией поиска.

## 📋 Функционал
- **Просмотр коллекции** картин с детальной информацией
- **Админ-панель Django** для управления контентом
- **Поиск по картинам** по названию и художнику
- **Адаптивный дизайн** в стилистике художественной галереи
- **Минималистичный интерфейс** с элегантной типографикой
- **Наследование шаблонов** через `{% extends 'base.html' %}` 
- **Единая навигация** на всех страницах
- **Базовый шаблон** с общими стилями
- **Футер** с информацией о правах

## 🚀 Установка и запуск

### 1. Клонирование репозитория
\`\`\`bash
git clone https://github.com/allegroandanteadagio-ship-it/Pre-Raphaelite-gallery.git
cd Pre-Raphaelite-gallery
\`\`\`

### 2. Создание виртуального окружения
\`\`\`bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate
\`\`\`

### 3. Установка зависимостей
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### 4. Настройка базы данных
\`\`\`bash
python manage.py migrate
\`\`\`

### 5. Создание суперпользователя (опционально)
\`\`\`bash
python manage.py createsuperuser
\`\`\`

### 6. Запуск сервера
\`\`\`bash
python manage.py runserver
\`\`\`

### 7. Открыть в браузере
- Главная страница: http://127.0.0.1:8000/
- Картины: http://127.0.0.1:8000/artworks/
- Художники: http://127.0.0.1:8000/artists/
- Админ-панель: http://127.0.0.1:8000/admin/

## 🎨 Модели данных

### Художник (Artist)
- Имя и фамилия
- Годы жизни
- Страна происхождения
- Биография

### Картина (Artwork)
- Название
- Художник (ForeignKey)
- Год создания
- Техника исполнения
- Размеры
- Местонахождение
- Описание
- URL изображения

## 🔍 Поиск
- Поиск по названию картины
- Поиск по имени художника
- Минималистичный интерфейс с иконкой лупы в поле ввода
- Сообщения о результатах поиска

## 🛠 Технологии
- **Backend:** Python 3.x, Django 5.2.9
- **Frontend:** HTML5, CSS3
- **База данных:** SQLite3
- **Безопасность:** python-dotenv для хранения секретов
- **Шрифты:** Google Fonts (Amarante)
- **Шаблоны Django:** наследование (`extends`), блоки (`block`)
- **CSS3:** Flexbox, CSS-переменные
- **HTML5:** семантические теги (`<nav>`, `<footer>`)

## 🏗️ Архитектура шаблонов

Проект использует систему наследования шаблонов Django. 
Все страницы наследуют общую структуру из `base.html`, 
что обеспечивает единый дизайн и простоту поддержки.
