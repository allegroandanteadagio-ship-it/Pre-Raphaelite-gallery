# 🎨 Галерея картин прерафаэлитов

Django-веб-приложение для отображения коллекции картин прерафаэлитов с адаптивным дизайном и административной панелью.

## 📋 Функционал
- **Просмотр коллекции** картин с детальной информацией
- **Админ-панель Django** для управления контентом
- **Адаптивный дизайн** в стилистике художественной галереи
- **Динамическая загрузка** изображений с внешних источников
- **Полнотекстовые описания** картин и художников

## 🚀 Установка и запуск

### 1. Клонирование репозитория
\`\`\`bash
git clone https://github.com/allegroandanteadagio-ship-it/Pre-Raphaelite-gallery.git
cd Pre-Raphaelite-gallery
\`\`\`

### 2. Создание виртуального окружения
\`\`\`bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate
\`\`\`

### 3. Установка зависимостей
\`\`\`bash
pip install -r requirements.txt
\`\`\`

### 4. Настройка базы данных
\`\`\`bash
python manage.py migrate
\`\`\`

### 5. Создание суперпользователя (опционально)
\`\`\`bash
python manage.py createsuperuser
\`\`\`

### 6. Запуск сервера
\`\`\`bash
python manage.py runserver
\`\`\`

### 7. Открыть в браузере
- Главная страница: http://127.0.0.1:8000/
- Картины: http://127.0.0.1:8000/artworks/
- Админ-панель: http://127.0.0.1:8000/admin/

## 🎨 Модели данных

### Художник (Artist)
- Имя и фамилия
- Годы жизни
- Страна происхождения
- Биография

### Картина (Artwork)
- Название
- Художник (ForeignKey)
- Год создания
- Техника исполнения
- Размеры
- Местонахождение
- Описание
- URL изображения

## 📁 Шаблоны

1. **home.html** - Главная страница
2. **artists.html** - Список художников
3. **artworks.html** - Коллекция картин (основной шаблон)

## 🛠 Технологии
- **Backend:** Python 3.x, Django 5.2.9
- **Frontend:** HTML5, CSS3
- **База данных:** SQLite3
- **Безопасность:** python-dotenv для хранения секретов
